## 📁 What's Inside

Practice scripts and mini-projects organized by topic — covering core Python concepts as the course progresses. Each file or folder focuses on one idea, written and tested hands-on.

Topics include (and keep growing):

- Functions — args, kwargs, closures, lambdas
- Comprehensions — list, dict & set
- Decorators & Generators
- Linear Data Structures
- Dunder / magic methods
- Built-ins like `zip()`, `enumerate()`, walrus operator `:=`
- Mini projects like seat booking, encoders, discount logic & more

## 🚀 How to Run

Make sure Python 3.x is installed, then:

```bash
# Clone the repo
git clone https://github.com/Sameer0726152/Python-Practice.git
cd Python-Practice

# Run any file
python filename.py
```
